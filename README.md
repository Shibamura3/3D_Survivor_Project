# 3D_Survivor

## 概要
DirectXを用いた3Dサバイバーライクゲームです。
大量の敵キャラクターが出現する中での動作安定性と、多様なプレイ環境への対応に注力して開発しました。

## 技術スタック
- 言語：C++ / HLSL
- ライブラリ：DirectX 11 (または12), SDL2
- ツール：Visual Studio 2022

## 主な実装機能
パフォーマンス最適化に加え、以下の機能を自力で実装しました。

- **高度な当たり判定システム**
  - 単純な球体判定だけでなく、Capsuleやレイキャストを用いた判定ロジックを自作。
- **マルチデバイス対応 (SDL2活用)**
  - SDL2を導入し、XboxやPS用コントローラーなど、多種多様な入力機器に対応。プレイヤーの環境を選ばない操作性を実現しました。
- **継続プレイを促すゲームサイクル**
  - **実績システム**: CSVによるデータ管理を導入。長期的なプレイ目標（実績解除）を提供し、拡張性も確保しました。
  - **ランキングシステム**: 展示会などの短時間プレイでも競い合えるよう、スコアランキング機能を実装しました。

## 特に注力した点：パフォーマンス最適化
- **大量オブジェクトの描画最適化**: GPUインスタンシングを導入し、FPSを20から60へ改善。
- **処理スパイクの解消**: 軌跡エフェクトの更新頻度最適化と描画コールのバッチ化により、瞬間的なFPS低下（1桁台）を根絶。

## 実行方法
1. 本リポジトリをクローンまたはダウンロード
2. `実行ファイル` フォルダ内の `game.exe` を実行
