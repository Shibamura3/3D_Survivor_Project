１．CRI ADX LE SDK for Native について


　CRI ADX LE SDK for Native は、オーディオミドルウェア CRI ADX の一部機能を Windows アプリケーション開発で使用するための無償版 SDK です。
　本SDKには、CRI ADX LE ライブラリと、関連するドキュメントファイルが同梱されています。

２．目的別の導入ガイド


　本SDKにはいくつかのドキュメントが収録されています。ユーザーの目的別に最初にどこを読めば良いかを紹介します。

（１）CRI ADX LE ライブラリでどんな事が出来るか知りたい人

　WEB公開しているマニュアル内の「ADX2の紹介」をご覧ください。
　このマニュアルには CRI ADX LE ライブラリには含まれない機能 (主にムービ再生機能) に関する記述も含みます。

（２）CRI ADX LE ライブラリをプログラムに組み込みたい人

　マニュアルの「導入の手引き」をご覧ください。
　具体的な使用方法は「機能説明」をご覧ください。

（３）CRI ADX LE ライブラリ用にサウンドデータを作ってみたい人

サウンドデータの作成には専用オーサリングツール CRI Atom Craft が必要です。
このツールは本 SDK パッケージには同梱されていません。専用のダウンロードURLから入手してください。
ツールの使用方法詳細はツールマニュアルをご覧ください。
